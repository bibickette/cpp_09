# Project presentation - `cpp_09`
## **Description**
This module focuses on practical **algorithm implementation and data manipulation** in C++.

The exercises introduce **file handling**, **mathematical expression evaluation**, and **efficient sorting algorithms**, reinforcing problem-solving skills and algorithmic thinking.

* * *

## Languages & Technologies

**Language**
- C++ (C++98 standard)

**Technologies**
- Makefile

* * *
## Key Concepts
- File input/output
- Data parsing
- Mathematical expressions (Reverse Polish Notation)
- Sorting algorithms and performance optimization
- Standard Template Library (STL) containers
- Exception handling

* * *
## Exercices Overview
### ex00. Bitcoin Exchange

File handling and data parsing exercise:
- Implement a program to read and process historical Bitcoin exchange data
- Compute and display results based on input values
- Handle invalid or malformed input gracefully

### ex01. Reverse Polish Notation

Mathematical expression evaluation:
- Implement a Reverse Polish Notation (RPN) calculator
- Support basic operators (`+`, `-`, `*`, `/`)
- Handle exceptions for invalid operations or malformed expressions

### ex02. PmergeMe

Sorting algorithm and performance optimization:
- Implement a sorting algorithm for integer sequences
- Use STL containers and optimize for efficiency
- Measure performance for large datasets

* * *

## Compilation rules

All exercises are compiled using the following flags :
- `-Wall`
- `-Wextra`
- `-Werror`
- `-std=c++98`

# Using `cpp_09`

## How to use `cpp_09`

*Note : this repository contains only the `cpp_09` module. It is part of the global [cpp_modules](https://github.com/bibickette/cpp_modules) repository, which regroups all C++ modules.*

1. Clone `cpp_09` in a folder first  : `git clone git@github.com:bibickette/cpp_09.git`
2. Move into the repository : `cd cpp_09`
3. Compile the module : `make`; this will compile all exercises contained in the module
4. You can now move into each exercices and run the corresponding executable
